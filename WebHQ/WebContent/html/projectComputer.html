[ Home Computer ]

I have always wanted to build my own computer. Not only have I always had a passion for all things tech, but I have also been an avid gamer since a very early age. Combine those 2 things and a gaming machine offspring was not far off. In February 2016 I finally got the chance to do it.

The deciding factor was when I got a Samsung Gear VR headset. One of the first things I tried to do was deploy my own VR app and quickly found out that my 7 year old dell computer just couldn't handle the graphics required to do so. I had always been able to limp my machine along with a new part here and there, but a 32 bit motherboard just wouldn't cut it any longer. And hey, if I'm replacing the motherboard then I'll need new RAM. If I'm getting new RAM then I should get a new ... and so on.

So here we are.


I couldn't be happier! Now my machine can handle basically everything I throw at it and keep running strong. I got to learn a lot as well. There is a big difference between having some book smarts about CPU socket types & RAM clock speeds and actually getting those things to cooperate in the real world. I had a lot of success on my first build with essentially no problems and I think the key to that was taking my time to really read and understand all the specifications of each product. Once I knew that everything would be compatible, assembly was not a problem at all.
